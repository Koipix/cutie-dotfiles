### Installation
```bash
git clone https://github.com/Koipix/cutie-dotfiles.git
cd cutie-dotfiles/script
```

### Apply Rice
While on `./script/` directory
```
./mklink.sh
```

### Restorice
If you want to restore to your own rice setup
```
./mkrestore.sh
```
